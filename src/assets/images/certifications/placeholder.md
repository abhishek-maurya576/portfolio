# Certification Images

Place your certification images in this folder with the following names:

- microsoft-ai-challenge.png
- girlscript-gsoc.png
- java-cert.png
- postman-cert.png
- c-programming-cert.png

Recommended image specifications:
- Format: PNG or JPG
- Resolution: 800x600 pixels (minimum)
- File size: Keep under 500KB for optimal loading
- Aspect ratio: 4:3 or 16:9
